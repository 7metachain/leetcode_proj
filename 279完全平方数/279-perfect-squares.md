# 279. 完全平方数

给定整数 `n`，返回和为 `n` 的完全平方数的最少数量。

- 完全平方数：等于某个整数平方的数，例如 `1, 4, 9, 16` 是完全平方数，`3, 11` 不是。
- 约束：`1 <= n <= 10^4`。

示例：

- 输入 `12`，输出 `3`，因为 `12 = 4 + 4 + 4`。
- 输入 `13`，输出 `2`，因为 `13 = 4 + 9`。
